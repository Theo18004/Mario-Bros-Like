# Marios Bros Like/ Platformer 2D

Fonctionnalités

  Déplacements : Le joueur peut courir et sauter (gestion de la gravité).

  Carte : Chargement de niveaux basés sur des tuiles (Tilemap).

  Collisions : Gestion des collisions avec le sol, les murs et les plateformes.

  Ennemis : Ennemis simples avec déplacements automatiques.

  Game Loop : Gestion fluide du temps et de l'affichage.

Technologies utilisées

  Langage : C

  Bibliothèque graphique : SDL2 (Simple DirectMedia Layer)

  Outils : GCC, Make
