◆ loadMapFromImage()

void loadMapFromImage ( const char * filename,
SDL_Rect ** platforms,
int * count,
SDL_Rect * playerPos )

Charge une carte à partir d'une image bitmap.

  • Cette fonction analyse chaque pixel d'une image :
  • Un pixel noir (RGB 0,0,0) génère une plateforme SDL_Rect.
  • Un pixel bleu (RGB 0,0,255) définit la position de départ du joueur.
  • Paramètres
    filenameChemin du fichier image (ex: "assets/map.png").
    platformsPointeur vers le tableau de rectangles qui sera alloué dynamiquement par la fonction.
    countPointeur vers un entier pour stocker le nombre de plateformes créées.
    playerPosPointeur vers un SDL_Rect pour mettre à jour la position initiale du joueur.
  • Note
    Il incombe à l'appelant de libérer la mémoire allouée pour platforms avec free().
    Avertissement
    Si l'image n'est pas trouvée ou si l'allocation échoue, un message d'erreur est affiché sur stdout.
